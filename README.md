# AntmanRL Coaching

A modern, responsive Rocket League coaching landing page built with React, Vite, and Tailwind CSS.

## Run locally

1. Install Node.js 18 or newer.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

## Project structure

## Notes

- Tailwind utilities handle the visual styling across the site.
- The contact form is UI-only and ready for future backend or Stripe integration.
- Social links and business copy can be replaced with live production content.
